Homework 1 - FizzBuzz
-------------

This directory is a scaffolded example of an assignment repository, it is used to demonstrate and test the checklist package.

## Task 1 - Implementation

Write a function `fizzbuzz()` that takes a numeric vector and prints, one value per line, `Fizz` for multiples of 3, `Buzz` for multiples of 5, `FizzBuzz` for multiples of both, and the value itself otherwise. Invalid inputs (non-numeric, negative, non-finite, or non-integer values) should produce an error.

## Task 2 - Validation

Show that your function produces the correct output (or a useful error message) for the good and bad inputs provided in `hw1.qmd`.

## Submission

Complete both tasks in `hw1.qmd`, then commit and push your work to this repository.
